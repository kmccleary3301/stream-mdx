# Post-100 Roadmap

This document is intentionally **not** part of the active completion denominator.

It exists to keep future ambitions visible without pretending they are still required for the current "finish the plan" bar.

## What is already in the active surfaced set

The active public content set is considered complete once these surfaces are present and maintained:

- docs landing + role tracks
- benchmark methodology and comparison surfaces
- demo page
- showcase entries for:
  - feature catalog
  - HTML overrides
  - custom regex plugins
  - MDX components
  - Mermaid diagrams
  - perf harness
  - terminal protocol flow
  - hosted worker deployment

## What moved out of the active denominator

These items are still valuable, but they are no longer required to call the current plan complete:

- a much larger 20+ article expansion program
- full site redesign against the earliest highest-ambition wireframes
- broader benchmark essay/report writing beyond the current methodology and comparison surface
- deeper polished reference apps beyond the minimal TUI example and the current docs site
- additional showcase families that do not expose a genuinely new product capability

## Rule

If new work is added from this roadmap, it should be tracked as a new program or milestone rather than silently expanding the denominator of the current one.
