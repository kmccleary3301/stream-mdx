# Math Inline Supported

Top-level inline math should anticipate cleanly: $x^2 + y_i$ with trailing prose.

- List item with fraction math: $\frac{a}{b}$ and text after it.
  - Nested item with root math: $\sqrt{x}$ and more trailing prose.

> Blockquote with script operators: $z_i^2$ and trailing content.
