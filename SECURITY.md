# Security Policy

## Reporting a vulnerability

Please do **not** open a public issue for security vulnerabilities.

Instead, use GitHub Security Advisories for this repository:

- Go to the repoâ€™s **Security** tab
- Click **Report a vulnerability**

If you are unable to use GitHub advisories, open a minimal issue and explicitly request a private contact channel (do not include exploit details).

## Supported versions

StreamMDX is currently in `0.x`. Only the latest published version is supported for security fixes.
